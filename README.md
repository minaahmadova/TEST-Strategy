# Kapital Bank 
TEST STRATEGİYASI
Testin məqsədi
- İstifadəçilərin saytda qeydiyyatdan keçə bilməsi, login/logout funksiyaların işləməsi, saytda olan dəyişikliklərin mövcud funksiyalar üzərindəki dəyişikliklərinin tapılması, ui/ux dizaynın istifadəçi rahatlığına xitab etməsi,müxtəlif browserlərdə saytın işləməsi
Test ediləcək funksiyalar
- Qeydiyyat
     İstifadəçilərin valid məlumatlar daxil edərək qeydiyyatdan keçəbilməsi
- Şifrə bərpası
     Unudulan passvordu bərpa edə bilmək
- Ödəniş
     İstifadəçilərin ödənişlərinin uğurla yerinə yetirilməsi
- Balansın yoxlanılması
     İstifadəçi balansındakı pulu görə bilir
- Təhlükəsizik
     İstifadəçi 2FA dən istifadə edə bilir  
- Bildirişlər
     İstifadəçiyə hesabındakı dəyişikliklərdən bildiriş gəlir (sms, kartdan pul çıxma, pul əlavə olma və s.)
